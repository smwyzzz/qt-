10-10-2

智能登录与用户注册模拟器
目标：模拟一个智能登录界面，结合行编辑器和组合框实现输入验证与自动补全。
功能要求：
1. 使用 QLineEdit实现以下三个输入框：
    (1)用户名框：正常显示模式，并设置 QCompleter提供历史用户名提示（例如："user2024", "admin", "guest"）。
    (2)密码框：设置为密码显示模式（黑点显示）。
    (3)年龄验证框：使用 QIntValidator限制输入必须为18至100之间的整数。
2. 添加一个 QComboBox用于选择用户角色（如："管理员", "普通用户", "访客"），并利用 QMap为每个角色关联一个权限等级值（例如：管理员->10，普通用户->5，访客->1）。
3. 实时反馈：将每个 QLineEdit的输入内容（密码框可显示为星号）和 QComboBox的当前选项，实时显示在对应的 QLabel中。

10-10-3

目标：创建一个动态的产品配置界面，重点考察组合框与数字选择框的联动。
功能要求：
1.使用两个 QComboBox：
    (1)产品类别下拉框：通过 addItem()逐个添加类别（如："电子产品", "文具", "食品"）。
    (2)具体产品下拉框：其内容根据选择的“产品类别”动态变化（例如：选择"电子产品"时，产品列表为"手机", "笔记本电脑"；选择"文具"时，列表为"钢笔", "笔记本"）。建议使用 QMap<QString, QStringList>来存储类别与产品的对应关系。
2.根据选择的“具体产品”，动态显示对应的配置选项：
    (1)如果选择"手机"，显示 QSpinBox用于选择内存大小（如：4GB, 8GB, 12GB）。
    (2)如果选择"钢笔"，显示 QComboBox用于选择墨水颜色（如："黑色", "蓝色", "红色"）。
    (3)如果选择"食品"，显示 QDateEdit用于选择保质期。
3.添加一个 QDateTimeEdit用于记录配置的创建时间。
4.所有配置最终点击“生成报告”按钮后，在一个QLabel或QTextEdit中生成格式化的产品配置摘要。
